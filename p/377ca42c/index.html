环境搭建

你已经有了一台 ECS，那么如何搭建 Wordpress 博客网站？

本站通过手工安装的方式搭建开源组合 LAMP 环境：部署LAMP环境（CentOS 7） (aliyun.com) 其实也可以按照阿里云的教程使用 WordPress 镜像进行建站。

Wordpress 的安装部署：手动搭建WordPress（CentOS 7） (aliyun.com) 包含设置域名、域名解析等步骤。

如果 ICP 未备案完成，WordPress 中的设置中的站点地址和 Wordpress 地址为服务器的公网 ip。ICP 备案成功后再同时改过来。

以下这种打开 WordPress 自定义时出现的错误就是 WordPress 地址和站点地址不一样导致的：wordpress无法动态预览更改 显示Non-existent changeset UUID.-问答-阿里云开发者社区-阿里云 (aliyun.com)

其他内容

SEO

在 wordpress，可以安装一款 SEO 插件，由它可指导关键词的安排与文章写作。然后向各搜索引擎提交网站的收录。

如何让百度快速收录WordPress网站？_小柯2319184669的博客-CSDN博客

What is an XML sitemap and why should you have one? • Yoast

Wordpress 自动更新失效

一个排查方向：登录后台>工具>站点健康，会有 wordpress 不能正常更新的说明。我的原因主要是 wordpress 没有权限写入文件夹内容。chmod -R 777 <文件夹> 就好了。

腾讯云 COS 工具

大坑！使用腾讯云 COS 工具将附件上传到 COS 后（一般包括你的主体样式），找不到方法逆转。这意味着你删掉 COS 中的文件后，即使服务器保留有，也不懂有什么办法逆转。（暂时找不到办法）
