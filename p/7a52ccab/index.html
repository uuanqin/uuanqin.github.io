GIF 动图简单制作

发表于2023-05-01|更新于2024-10-02|工具箱
|总字数:1.2k|阅读时长:3分钟|浏览量:|评论数:

想自己组合一个 GIF，本来以为这种简单的东西交给在线网站做就行了，不必用到 PR 来做，不然有点大材小用的感觉。但是在互联网找一圈后，能成功达到我的要求的根本没有，主要槽点在于：

  • 没有符合我需求的 GIF 处理。我要的是画面上的拼接，不是 GIF 前后合并
  • 不能直击痛点。WPS 图像处理也有 GIF 编辑，但是功能少和简陋。一些网站提供图像拼接，但是只能在规定的框架格式下运行（还要关注公众号）
  • 不能逐帧操作。WPS 图像处理可以逐帧操作，但简陋，不好调整图片
    额…突然发现我的需求还是挺复杂的。那么，刁钻的要求往往需要专业的软件解决，于是我就打算使用 PR 完成这个需求。

其实到最后，PR 也没能一键解决最后的需求。整个流程使用了许多软件和第三方服务，下面我将按照处理步骤分块拆解，读者可自由组合流程达到最终的目的。

需求描述

我拥有 GIF 图一张，PNG 素材两张（两个气泡框），我需要把他们整合到同一个画面生成 GIF，并且最后生成的 GIF 不同的帧显示指定的 PNG。

如果你成功导出透明背景的 GIF 了，那么这篇文章就结束了！
