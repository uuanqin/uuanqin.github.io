使用 Adobe Premiere Pro 实现 GIF 动图画面拼接

发表于2023-05-01|更新于2023-10-20|工具箱
|字数总计:790|阅读时长:2分钟|阅读量:

想自己组合一个 GIF，本来以为这种简单的东西交给在线网站做就行了，不必用到 PR 来做，不然有点大材小用的感觉。但是在互联网找一圈后，能成功达到我的要求的根本没有，主要槽点在于：

  • 没有符合我需求的 GIF 处理。我要的是画面上的拼接，不是 GIF 前后合并
  • 不能直击痛点。WPS 图像处理也有 GIF 编辑，但是功能少和简陋。一些网站提供图像拼接，但是只能在规定的框架格式下运行（还要关注公众号）
  • 不能逐帧操作。WPS 图像处理可以逐帧操作，但简陋，不好调整图片
    额…突然发现我的需求还是挺复杂的。那么，刁钻的要求往往需要专业的软件解决，于是我就打算使用 PR 完成这个需求。
    其实到最后，PR 也没能一键解决最后的需求。

需求描述

我拥有 GIF 图一张，PNG 素材两张（两个气泡框），我需要把他们整合到同一个画面生成 GIF，并且最后生成的 GIF 不同的帧显示我指定的 PNG。

导出

在老版本的 PR 里，GIF 的导出不支持把 Alpha 通道导出，也就是导出的 GIF 是黑色背景的。你可以看这篇文章确定一下你的 PR 是否可以直接导出透明背景的的 GIF：Premiere 2022 直接导出带透明通道 GIF 动画 - 知乎 (zhihu.com)，注意，导出格式选择动画 GIF，不是GIF！

如果你成功导出透明背景的 GIF 了，那么这篇文章就结束了！
