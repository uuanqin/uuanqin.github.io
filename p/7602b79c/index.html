整除

如果 aaa 和 bbb 是整数，且 a≠0a≠0a=0，如果有整数 ccc 使得 b=acb = acb=ac，或者 b/ab/ab/a 是一个整数，我们称 a 整除 b。

  • aaa 是 bbb 的一个因子（或除数，约数）。bbb 是 aaa 的一个倍数。
  • 记号 a∣ba\mid ba∣b 表示 a 整除 b，a∤ba\not\mid ba∣b 表示 aaa 不能整除 bbb。

0 是所有非 0 整数的倍数。对于整数 b≠0b\ne 0b=0，b 的约数只有有限个。

定理 TY-1

a，b，c 是整数且 a≠0a≠0a=0，

  1. 如果 a∣ba \mid ba∣b 且 a∣ca \mid ca∣c ，那么 a∣(b+c)a \mid (b + c)a∣(b+c)。同时也说明，如果 a 同时整除 b 和 c，那么它也必须整除 b 和 c 的差。
  2. 如果 a∣ba \mid ba∣b ，那么对于任意整数 c 都有 a∣bca \mid bca∣bc。
  3. 如果 a∣ba \mid ba∣b 且 b∣cb \mid cb∣c ，那么 a∣ca \mid ca∣c。
小练习：证明任意相邻的两个奇数互质（互素）

一些推论 TY-2

  1. 如果 a，b，c 是整数且 a≠0a≠0a=0，满足 a∣ba \mid ba∣b 和 a∣ca \mid ca∣c，当 m 和 n 是整数时，有 a∣mb+nca \mid mb + nca∣mb+nc 。
  2. a∣b  ⟺  −a∣b  ⟺  a∣−b  ⟺  ∣a∣∣∣b∣a\mid b\iff -a\mid b \iff a\mid -b\iff |a| \mid |b|a∣b⟺−a∣b⟺a∣−b⟺∣a∣∣∣b∣
  3. a∣b∧b∣a  ⟹  b=±aa\mid b \land b\mid a \implies b=\pm aa∣b∧b∣a⟹b=±a
  4. 设 m≠0m\ne 0m=0，那么 a∣b  ⟺  ma∣mba\mid b \iff ma \mid mba∣b⟺ma∣mb
  5. 设 b≠0b\ne 0b=0，那么 a∣b  ⟹  ∣a∣≤∣b∣a\mid b \implies |a| \le |b|a∣b⟹∣a∣≤∣b∣
  6. 设 a≠0,b=qa+ca\ne 0,b=qa+ca=0,b=qa+c，那么 a∣b  ⟺  a∣ca\mid b \iff a\mid ca∣b⟺a∣c

带余数除法

除法算法：如果 aaa 是一个整数且 ddd 是一个正整数，那么存在唯一整数 qqq 和 rrr，其中 0≤r<d0 ≤ r < d0≤r<d, 使得 a=dq+ra = dq + ra=dq+r。

  • ddd 称为除数
  • aaa 称为被除数
  • qqq 称为商
  • rrr 称为余数。在这里的定义下，余数 rrr 是最小非负余数。

有：

q=a div dr=a mod d\begin{aligned} q=&a~ \textrm{div} ~d \\ r=&a \bmod d \end{aligned} q=r=​a div damodd​

小学数学常识之中学语文知识：「除」和「除以」

6÷26÷26÷2 读作：「六除以二」，也就是「（以）二除六」。在后一句中，主语是省略的（或者说主语就是你/我），谓语是「除」这个动作，宾语是作为分子的「六」，而「以二」则是状语。前一句是后一句的倒装。

同余

如果 a 和 b 为整数且 m 为正整数，则当 m 整除 a−ba - ba−b 时称 a 模 m 同余 b。

  • a 模 m 同余 b，记为 a≡b ( mod m)a\equiv b ~(\bmod m)a≡b (modm)
  • a 和 b 不是模 m 同余的，记为 a≢b ( mod m)a\not\equiv b ~(\bmod m)a≡b (modm)

「同余」有时也称「等价」。

定理 TY-3：令 a 和 b 是整数，m 是正整数，则 a≡b ( mod m)  ⟺  a mod m=b mod ma\equiv b ~(\bmod m)\iff a \bmod m = b \bmod ma≡b (modm)⟺amodm=bmodm

ME：同余。顾名思义，就是两者之间都具有相同余数。

    • 如果 a≡b( mod m)a \equiv b(\bmod m)a≡b(modm) 那么 c+a≡c+b( mod m)c + a \equiv c + b(\bmod m)c+a≡c+b(modm)，其中 c 是任意整数。

    • 注意：将同余除以整数并不总是产生有效的同余。后文将会探讨。

设 m 是正整数且 a 和 b 是整数，那么:

  • (a+b) mod m=((a mod m)+(b mod m)) mod m(a + b) \bmod m = ((a \bmod m) + (b \bmod m)) \bmod m(a+b)modm=((amodm)+(bmodm))modm

定理 TY-10：如果 a 和 m 为互素的整数且 m>1，则 a 模 m 的逆存在。而且这个模 m 的逆是唯一的。

即存在唯一小于 m 的正整数 ā 是 a 模 m 的逆，并且 a 模 m 的其他每个逆均与 ā 模 m 同余。（ME：注意这里的「唯一」的意思）。

证明：

  1. 存在性。
  • 由于 gcd(a，m)=1，根据贝祖定理 ZS-5，存在整数 s 和 t，使得 sa+tm=1sa+tm=1sa+tm=1。因此，sa+tm≡1 ( mod m)sa+tm≡1~(\bmod m)sa+tm≡1 (modm).

  1. 唯一性。
