Nginx 配置

由于之前[1]弄了好一会这个配置，为的是网站实现全程 HTTPS 访问。现在我也想让外部访问朋友圈接口时也使用 HTTPS。

通过 location 正则匹配，识别出/fc_backend/开头的链接为朋友圈 API 的请求，并将其转发到端口 8000（朋友圈默认端口）。发送给 8000 端口的地址不包含/fc_backend/。

当然，这里需要前端也要配合发出/fc_backend/开头的请求。

修改配置文件后别忘了重启 Nginx。
